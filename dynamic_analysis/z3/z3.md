## Getting started
```py
x = Int('x')
y = Int('y')
# == x, y = Ints('x y')
solve(x > 2, y < 10, x + 2*y == 7)
```

### Variable types
- z3 data type

```py
Int('x')
Real('x')
BitVec('x', 16)
```
- boolean

```py
Bool('p')
And()
Or()
Not()
Implies()
If()
```
- z3 value

```py
IntVal(1)
RealVal(1)
BitVecVal(10, 32)
```
### Set option

## Solver
```py
hmm
```

### result
- sat

if a solution was found.
- unsat

if no solution exists.
- unknown

fail to solve a system of constraints. (non-polynomial, `2**x`)

### model
A model is an interpretation that makes each asserted constraint true
